# Projecte: Invoca la Bogeria

Benvingut a la documentació oficial d'aquesta peça de programació d'avantguarda. Si has arribat fins aquí, és que busques respostes que potser la teva ment no pot processar.

## Funcionament
L'aplicació és d'una simplicitat brutal i minimalista:
1. L'usuari prem el botó **"Invoca la bogeria"**.
2. Un algorisme d'alta precisió (un `Math.random()`) selecciona una perla de saviesa absurda del nostre repositori de contingut.
3. La frase apareix en pantalla per qüestionar la realitat de l'usuari.

## Stack Tecnològic
* **HTML5**: Estructura òssia de la bogeria.
* **JavaScript (Vanilla)**: El sistema nerviós que tria frases a l'atzar.
* **JSON**: El magatzem d'idees sense sentit.

## ⚠️ Advertència
L'ús excessiu d'aquest botó pot provocar rialles espontànies o una profunda confusió existencial. No ens fem responsables de la pèrdua de lògica durant el seu ús.
